git commands

  • git init: initializes empty git repo (master branch)
  • git add README.md: creates/adds a README file
  • git add .: adds all files in the current directory
  • git commit -m "first commit": commits
  • git remote add origin https://github.com/katkim0307/"repo name".git
  • git pull origin master: if the README.md is already created in the repo, pull it before push
  • git push -u origin master: push the repo
  • git push origin master --force: force pushing the repo
  • git remote -v: lists the branch url
  • git status: status of the used branch
*** If you need to remove the git reference for whatever reason, you can use the command rm -rf .git/
